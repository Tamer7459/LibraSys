# 📚 LibraSys – Library Management System

### 🚀 Simple, Structured & Scalable Library Management Web App

**Description:** A web-based library management system built with Django that allows efficient management of books, users, and library operations through a structured backend and clean interface.

---

## 📌 Project Overview

This project is a **Library Management System (LibraSys)** designed to manage books, categories, and users in a structured way using Django framework.

It focuses on **backend development, database management, and MVC architecture (Model-View-Controller)**.

---

## 🚀 Features

* 📖 Manage books (add, update, delete)
* 👤 User management system
* 🗂️ Organized book categories
* 🔍 Search functionality for books
* 🧾 Admin dashboard (Django admin)
* ⚙️ Clean MVC architecture

---

## 🛠️ Technologies Used

* Python
* Django Framework
* HTML5
* CSS3
* SQLite (default database)

## 🧩 How It Works

1. Django handles routing and backend logic.
2. Models define database structure (books, users, categories).
3. Views control application logic and data processing.
4. Templates render dynamic HTML pages.
5. Admin panel manages all data efficiently.

## ▶️ How to Run

1. Clone the repository:

```bash id="lib-git001"
git clone https://github.com/Tamer7459/LibraSys.git
```

2. Navigate to the project:

```bash id="lib-cd001"
cd LibraSys
```

3. Install dependencies:

```bash id="lib-install001"
pip install -r requirements.txt
```

4. Run migrations:

```bash id="lib-migrate001"
python manage.py migrate
```

5. Start server:

```bash id="lib-run001"
python manage.py runserver
```

6. Open in browser:

```
http://127.0.0.1:8000/
```
